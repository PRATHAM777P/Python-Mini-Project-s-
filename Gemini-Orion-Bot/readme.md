# Gemini-Orion-Bot

**Gemini-Orion-Bot** is a modern, web-based AI chatbot that combines the cosmic inspiration of two stellar constellations—**Gemini** and **Orion**—to deliver a futuristic, intelligent, and interactive experience. The name hints at **dual intelligence** (Gemini) and a **guiding star** (Orion), making it memorable and techy.  

This project showcases a friendly AI assistant with a sleek interface, light/dark mode, and interactive features.

---

## 🌟 Features

- **Dark/Light Mode Toggle** – Smooth transition between themes.
- **Floating Action Buttons (FABs)** – For sending prompts and attaching files.
- **Interactive Suggestions** – Hover and click animations for prompts.
- **Auto-Scrolling Chat Area** – Automatically scrolls to the latest messages.
- **Message Timestamps** – Optional enhancement to display when messages are sent.
- **Clear All Chats** – Includes confirmation to prevent accidental deletions.
- **Simulated AI Responses** – Interactive bot messages for a conversational experience.
- **Accessibility-Friendly** – Semantic HTML and ARIA labels included.
